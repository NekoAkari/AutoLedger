# AutoLedger
A simple iOS expense tracking app built with **SwiftUI**.

## Features
- Add income and expense records
- View transaction history
- Monthly summary of balance
- Simple and clean interface

## Tech Stack
- Swift
- SwiftUI
- SwiftData
- Xcode

## Project Structure
```
AutoLedger
├── Models
├── Views
├── Components
└── Utilities
```
## Development
Clone the repository:

```
git clone https://github.com/NekoAkari/AutoLedger.git
```

Open the project in **Xcode**:

```
AutoLedger.xcodeproj
```

Run the project on the iPhone simulator.
